# Guides

## Add a top-movies RSS feed to put.io

1. Open the home page and click the RSS icon next to the top-movies source selector.
2. Copy the generated feed URL.
3. In put.io, add that feed as an RSS subscription.
4. Keep the link private, because it is generated for your account.

## Report a crash without automatic telemetry

1. If the crash fallback appears, add a short note about what you were doing.
2. Click `Copy report` if you want to paste the details manually.
3. Click `Create GitHub issue` if you want a prefilled bug report in the public repo.
